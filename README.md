# BMW Global Sales & Electrification Analysis

## Project Overview

This project analyzes **BMW global sales data (2018–2025)** to identify key market trends, regional performance, and the growing impact of electric vehicles on the automotive industry.

The goal of this project is to demonstrate a **data analysis workflow using Python**, including:

* Data cleaning
* Exploratory data analysis
* Business insights
* Data visualization

The analysis was conducted using **Python (Pandas, Matplotlib, Seaborn)** and the results are presented both through visualizations and an analytical notebook.

---

# Dataset

The dataset contains BMW global sales information including:

* Units sold
* Revenue
* Average vehicle price
* Regional sales distribution
* Electric vehicle (BEV) share
* Macroeconomic indicators such as GDP growth and fuel prices

Time period analyzed:

**2018 – 2025**

---

# Data Analysis Process

The analysis followed these steps:

### 1. Data Cleaning

* Checked for missing values
* Removed duplicates
* Verified data types
* Created time-based features for trend analysis

### 2. Exploratory Data Analysis

Exploration of:

* Global sales trends
* Regional performance
* Model popularity
* Electric vehicle adoption
* Economic correlations

### 3. Data Visualization

Key metrics were visualized to highlight trends and insights.
